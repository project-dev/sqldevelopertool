簡単な使い方説明

簡単に使い方を説明していきます。
  • データベースに接続する
  • SQLを実行する
  • テーブルの内容をコピーする
  • テーブルの内容をエクスポートする
  • DB再現SQLを作成する

データベースに接続する

データベースに接続する方法として以下の方法があります。
  1. データベースを接続後に選択する
  2. データベースを明示的に指定して接続する
まず、メニューから[ファイル]→[接続]を選択、または、ツールバーの[接続]ボタンを押下し、 ログインダイアログを表示します。
ダイアログの[新規]ボタンを押下し、ログインセットの名前を決めます。
ログインセットは、データベースにログインする為の情報のセットです。
ログインセットを作成しておくと、簡単にデータベースに接続することができます。
ログイン設定リストから、作成したログインセットを選択します。
接続情報の種別から任意の情報を選択してください。
  1. SQLServer/MSDE
    • SQLServer/MSDEのデータベースに直接接続します。
  2. ODBC接続
    • ODBCを使ってデータベースに接続します。
種別を選択したら、接続するサーバーを記述してください。
サーバーを記述したら、ユーザー名を記述してください。
ユーザー名を記述したら、パスワードを記述してください。

サーバー、ユーザー名、パスワードの記述が終わったら、接続するデータベースを設定します。
種別がSQLServer/MSDEの場合、ログイン後にDBを選択するか、明示的にDBを選択するかの２つの方法を選択できます。
接続先のDB名が分からなかったり、複数のデータベースにアクセスする可能性がある場合は[ログイン後、DBを選択する]をチェックしてください。
接続先のDB名がわかっている場合は、[DB/データソース]のところにデータベース名を記述してください。

ODBC接続の場合は、[DB/データソース]のところに、必ず接続するデータソース名を記述します。

接続するDB/データソース名を記述したら、ログイン設定の[変更]ボタンを押下して、記述した内容を保存します。
[OK]ボタンを押下すると、データベースに接続します。

SQLを編集、実行する

実行

SQLを実行するには、データベースに接続する必要があるので、データベースに接続してください。
接続できたら、[SQLタブ]をアクティブにしてください。
特に操作していない場合は、[SQL(1)]というタブがあると思います。

簡単なSQLを記述して実行します。
SELECT * FROM テーブル名
SQLを記述したら、メニューの[ファイル]→[実行]を選択、またはツールバーの[実行]ボタンを押下、または、[Ctrl+Return]を押下してください。
SQLに問題が無ければ、実行結果が[テーブル表示ビュー]に表示されます。

このように、SQLを実行していきます。

次に複数のSQLを実行します。
INSERT INTO テーブル名 VALUES(値)
SELECT * FROM テーブル名
実行すると、情報が挿入されて、実行結果が[テーブル表示ビュー]に表示されます。

SQLの整形

[Ctrl+Q]を押下してください。
SQLが整形整形されました。
SQLが長くなってしまった場合に、このように整形することで見やすくなります。

SQL補完機能

次に、”SEL”とSQLエディタに打ってください。
その後、[Ctrl+Space]を押下してください。
候補ウィンドウが表示されます。
候補ウィンドウから、”SELECT <colum(s)>”を選択し、リターンキーを押下してください。
”SEL”が”SELECT”に置き換えられます。

SQLの編集時にSQLの構文が知りたい場合などに利用してください。

テーブルの内容をコピーする

[テーブル表示ビュー]に何らかの情報を表示してください。
コピーしたい行を選択してください。
[Ctrl+C]を押下してください。
テキストエディタを起動して、貼り付けを行なってください。
選択した内容が貼り付けられます。

テーブルの内容をエクスポートする

エクスポートしたいテーブルを、情報ツリーのテーブルの中から選択します。
メニューの[ファイル]→[エクスポート]または、ツリーのポップアップメニューの[エクスポート]を選択します。
CSV形式とHTML形式の2つの形式を選択できるので、用途や好みに合わせて選択してください。
保存ダイアログが表示されるので、ファイル名などを記述して、保存ボタンを押下すれば、エクスポートが完了します。

DB再現SQLを作成する

[ツール]→[DB再現SQL生成]を選択します。
再現したいテーブルやストアドプロシージャをチェックしてOKボタンを押下します。
SQLの生成が終わると、”生成されたSQL”ダイアログが表示されます。
[Ctrl+C]でコピーするか、保存ボタンで保存してください。
